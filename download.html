Instrucciones para iOS (TestFlight)

Nuestra app para iOS está actualmente en TestFlight, lo que significa que estás accediendo a una versión de prueba.

  1. Descarga la aplicación TestFlight desde la App Store si aún no la tienes.
  2. Haz clic en el siguiente enlace desde tu dispositivo iOS para unirte a nuestro programa de pruebas: Unirse a TestFlight
  3. Sigue las instrucciones en TestFlight para instalar Together.
  4. Una vez instalada, podrás usar la app normalmente y recibirás actualizaciones a través de TestFlight.

Nota: Como beta tester, puedes encontrar algunos problemas o funcionalidades incompletas. ¡Tu feedback es valioso para nosotros!

Descarga para Android

Para instalar nuestra aplicación en tu dispositivo Android:

  1. Haz clic en el siguiente enlace para descargar el APK directamente: Descargar APK
  2. Es posible que debas permitir la instalación de aplicaciones de fuentes desconocidas en la configuración de tu dispositivo.
  3. Una vez descargado, abre el archivo APK para comenzar la instalación.

Pronto estaremos disponibles en Google Play. ¡Gracias por ser parte de nuestros primeros usuarios!
